Resumo:

Para o 5º trabalho prático da unidade curricular de Desenvolvimento de Aplicações Web é necessário criar um servidor que gera páginas HTML com recurso ao motor de templates Pug a partir de ficheiros JSON relativos a obras musicais.

Tal como acontecia para o trabalho prático anterior, o servidor utiliza o path contido no URL do pedido para encontrar a página pedida, se estiver em uso um caminho previsto. Todas as páginas servidas são geradas a partir de templates pelo motor Pug. As templates utilizam uma linguagem abreviada que se traduzirá eficazmente numa página HTML. O conteúdo dessa página poderá ser definido dinamicamente por objetos ou arrays passados à função Pug relativa a uma dada página (ou tipo de páginas).

À semelhança do 4º trabalho, existem novamente 2 tipos de páginas - índice e páginas sobre obras musicais. O índice é gerado fazendo uso dos nomes dos ficheiros JSON sobre as obras musicais, referenciando cada uma pelo seu identificador. Por sua vez, a geração de páginas sobre obras musicais tem em conta a possibilidade de ausência de determinados elementos que não são comuns a todos os ficheiros do seu tipo e o seu conteúdo, como seria de esperar, é apresentado adequadamente de acordo com a presença ou ausência desses possíveis elementos.
