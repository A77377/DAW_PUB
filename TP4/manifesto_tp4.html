Resumo:

No âmbito do 4º trabalho prático da unidade curricular de Desenvolvimento de Aplicações Web é pedido que seja escrito um servidor de páginas HTML geradas a partir de um dataset relativo aos Arqueossítios do Noroeste Português, contido num ficheiro XML.

O servidor a ser desenvolvido deve verificar o path contido no URL do pedido e devolver a página pedida, se o caminho espeficiado estiver previsto para uma das páginas a servir. Uma das páginas válidas a servir será a do índice, que apresenta uma lista ordenada dos concelhos onde se encontram arqueossítios do dataset. Cada elemento da lista terá a si associada, por sua vez, uma lista ordenada dos nomes dos arqueossítios no dado concelho. As restantes páginas válidas correspondem às páginas relativas a cada arqueossítio. Cada entrada para um arqueossítio no índice deve ter um link que gere um pedido ao servidor para obtenção da sua respetiva página. De modo semelhante, cada página relativa a um arqueossítio terá um link que permite pedir ao servidor a página HTML relativa ao índice.
